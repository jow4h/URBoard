## ğŸ§© Chrome Extension
URBoard can be used as a Chrome Extension to override your new tab page.

### Option 1: Download from Releases (Recommended)
1. Go to the [Releases](https://github.com/[username]/urboard/releases) page on GitHub.
2. Download the `urboard-extension.zip` file.
3. Extract the ZIP file to a folder on your computer.
4. Open Chrome and go to `chrome://extensions/`.
5. Enable **Developer mode**.
6. Click **Load unpacked** and select the extracted folder.

### Option 2: Local Installation (Development)
1. Open Chrome and go to `chrome://extensions/`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select the `extension` folder from the source code.
