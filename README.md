## ğŸ› ï¸ Installation

### Chrome / Edge / Brave
1. Download `urboard-chrome.zip` from the [Latest Release](https://github.com/jow4h/URBoard/releases/latest).
2. Extract the ZIP file to a folder.
3. Open `chrome://extensions` in your browser.
4. Enable **"Developer mode"** (top right).
5. Click **"Load unpacked"** and select the extracted folder.

### Firefox
You can install the official extension from the [Mozilla Add-ons Store](https://addons.mozilla.org/tr/firefox/addon/urboard/).
